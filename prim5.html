Пример 5: Использование датчика температуры и влажности DHT11

Код программы:

В данном примере используется цифровой входной пин Arduino для считывания данных с датчика DHT11. Функция dht.readTemperature() используется для считывания значения температуры, а функция dht.readHumidity() - для считывания значения влажности. Полученные значения выводятся на серийный монитор с помощью функции Serial.println().

Схема подключения:

В данной схеме датчик DHT11 подключается к выбранному цифровому входному пину Arduino. Питание и земля подключаются к соответствующим пинам Arduino.
