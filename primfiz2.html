Пример: Измерение уровня освещенности с Arduino

Код программы:

Данный пример демонстрирует использование фоторезистора для измерения уровня освещенности и вывода его значения через последовательный порт. Код программы инициализирует аналоговый пин A0 в качестве входа для считывания значения напряжения с фоторезистора. Затем происходит преобразование напряжения в уровень освещенности, и результат выводится через последовательный порт с помощью функции Serial.print().

Схема подключения:

В данном случае, фоторезистор подключается к аналоговому пину A0 на Arduino. Контакт одного конца фоторезистора подключается к 5V пину Arduino для обеспечения питания, контакт другого конца фоторезистора подключается к GND пину Arduino, а контакт между фоторезистором и резистором (например, 10к Ом) подключается к аналоговому пину A0. После подключения фоторезистора Arduino будет считывать напряжение с фоторезистора, которое затем будет преобразовано в уровень освещенности.
