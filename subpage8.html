Подробный справочник по Arduino IDE

Arduino IDE представляет собой интегрированную среду разработки (Integrated Development Environment) для программирования платформы Arduino. В данном справочнике вы найдете основные конструкции языка Arduino и инструкции по их использованию.

Условные операторы

Оператор if

Оператор if позволяет выполнять определенные действия, если указанное условие истинно:

if (условие) { // код, который будет выполнен, если условие истинно }
Оператор if...else

Оператор if...else позволяет выполнить определенный блок кода, если условие истинно, и другой блок кода, если условие ложно:

if (условие) { // код, который будет выполнен, если условие истинно } else { // код, который будет выполнен, если условие ложно }
Оператор else if

Оператор else if позволяет проверить дополнительные условия, если предыдущие условия не выполнились:

if (условие1) { // код, который будет выполнен, если условие1 истинно } else if (условие2) { // код, который будет выполнен, если условие1 ложно, а условие2 истинно } else { // код, который будет выполнен, если все условия ложны }

Циклы

Цикл for

Цикл for позволяет выполнять повторяющиеся действия определенное количество раз:

for (инициализация; условие; шаг) { // код, который будет выполняться в цикле }
Цикл while

Цикл while выполняет повторяющиеся действия, пока указанное условие истинно:

while (условие) { // код, который будет выполняться в цикле }
Цикл do...while

Цикл do...while выполняет повторяющиеся действия, пока указанное условие истинно. Отличие от цикла while в том, что код внутри цикла выполнится хотя бы один раз:

do { // код, который будет выполняться в цикле } while (условие);

Операторы управления

Оператор break

Оператор break используется для прерывания выполнения цикла:

for (int i = 0; i < 10; i++) { if (i == 5) { break; // прервать цикл при i равном 5 } }
Оператор continue

Оператор continue используется для перехода к следующей итерации цикла, минуя оставшуюся часть текущей итерации:

for (int i = 0; i < 10; i++) { if (i == 5) { continue; // перейти к следующей итерации при i равном 5 } }
